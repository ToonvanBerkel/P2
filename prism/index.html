HTML - Wat is HTML?

Html is de basisbouwsteen van het web. Het is een eenvoudige opmaaktaal die wordt gebruikt om webpagina's te maken. Stel je HTML voor als de 'skeletstructuur' van een pagina. Het geeft instructies aan de webbrowser over hoe de inhoud van een pagina moet worden weergegeven. HTML is gemakkelijk te leren, u zult ervan genieten!

Eerste HTML code:


                * {
                    margin: 0;
                    padding: 0;
                }
    
                section {
                    width: 100vw;
                    height: 100vh;
                    display: flex;
                    align-items: center;
                    justify-content: center;
                }
                
                .language-css{
                    width: auto;
                }

Eerste HTML code:

• De !DOCTYPE html verklaring definieert dat dit document een HTML5document is.
• Het html element is het hoofdelement van een HTML-pagina. De lang=”en” in de html element, geeft aan dat engels de taal is op de website.
• Het head element bevat meta-informatie over de HTML-pagina.
• Het meta is voornamelijk voor het aangeven wat er in de website staat en waar het over gaat, om hoger in het zoek systeem te komen. De charset=”UTF-8” in de meta specificeert een karakterset.
• Het title element specificeert een titel voor de HTML-pagina (die wordt weergegeven in de titelbalk van de browser of op het tabblad van de pagina).
• Het body element definieert de hoofdtekst van het document en is een container voor alle zichtbare inhoud, zoals koppen, alinea's, afbeeldingen, hyperlinks, tabellen, lijsten, enz.
• Het h1 element definieert een grote kop.
• Het p element definieert een paragraaf./p
